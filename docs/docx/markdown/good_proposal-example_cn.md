主题：在不同类型的 EC2 实例上对 Shadowsocks 代理服务器进行基准测试

项目成员：

项目性质：技术型

项目摘要：

Shadowsocks (SS) 是一种类似于 VPN 的网络代理工具，在中国大陆被广泛用于突破防火墙。Shadowsocks 是一个开源工具，其性能受 VPS 能力的影响。普遍的理解是，具有大带宽和强大硬件的 VPS 比基础 VPS 能够代理更多的数据包。在我们的项目中，我们将在不同类型的 EC2 实例上安装 Shadowsocks，并测量和评估其代理能力和性能。

我们将研究 Shadowsocks 的安全性。最近的报告表明，机器学习算法可能被用于识别网络中的 Shadowsocks 流量，这使得在防火墙上部署拒绝服务应用成为可能。我们将讨论 Shadowsocks 的安全性并探讨可能的对策。

Shadowsocks 本质上通过代理数据包来突破网络防火墙的限制。Shadowsocks 使用 chacha2.0 和 AES 加密算法来保护代理数据包，从而防止流量被拦截。此外，服务器端 TCP 转发的优化可以显著提高 SS 的吞吐量。我们将在 SS 服务器上部署 TCP 加速应用，并测试其性能与服务器硬件之间的关系。

我们将对 Shadowsocks 进行成本分析，并讨论不同类型的 AWS 实例适合的用户类型。这对中国大陆的大学生将非常有用，因为他们可以申请 AWS 学生账户并充分利用其信用额度。

整个压力测试和基准测试计划将在团队会议中讨论。为了确保不同的服务器位置和服务器类型都能通过预先设计的脚本进行测试。
